Search and use PicHit.Me photos in your posts and pages without ever leaving WordPress.

== Description ==

PicHit.Me is a global Market – A fun and vast photo market, where you get access to people’s photos and (in next update) can request the photos you need. The photo market’s goal is to be where the photo need is big, like here on WordPress.

The PicHit.Me WordPress plugin integrates seamlessly with your WordPress 3.5+ Media Manager and lets you create posts with photos without ever leaving your WordPress site.

**Features include:**

*	Search photos customized to your PicHit.Me account.

*	Download and preview images before you post.

*	Download, insert and publish any size image using PicHit.Me.

**Using the plugin once installed:**

*	Go to a new or existing post/page.

*	Select PicHit.Me above the composition area, or access PicHit.Me from the Media tab.

*	Start searching for images to add to your post. Login to your PicHit.Me account to download images for preview or publication, and to access your content and active images.

Not a PicHit.Me user yet? [Sign up here](http://pichit.me/signup/).

== Installation ==

1. Upload pichit-me to /wp-content/plugins/.

2. Activate plugin through the WordPress Plugins menu.

3. Click on “PicHit.Me” in settings, signup and authorize.

4. Click on "PicHit.Me" in edit post/page screen to browse the PicHit.Me library.
